Letterboxd Data Scraping (Personal Project)

Code Available on GitHub.

Letterboxd is a rating, reviewing, logging, etc. site for movies. You can also make a watchlist, keeping track of movies you want to see. I wanted to see watchlist movies in common with my friends, so I wrote a Python program to parse the public letterboxd pages with Beautiful Soup, and gather movies in common between people's watchlists.

Ideas for improvements I have are using a more efficient algorithm for finding commonality between lists, as well as integrating this into a useable web app.
